❤️ Dating Website : https://dating-website-stefano.netlify.app/

A fun dating website with one simple goal: get a date!

How it works:
1. Enter your name and email.
2. Choose an activity.
3. Select a date and time.
4. Send your request.

⚠️ Important:
You can click "Yes" anytime, but clicking "No" is a bit more challenging...
The "No" button has a habit of running away whenever you try to press it. 😄

Features:
- Interactive and responsive UI
- Animated moving "No" button
- Date request form validation
- Email notifications via Nodemailer
- Custom date and activity selection
- Mobile-friendly design

Tech Stack:
- React
- JavaScript
- HTML5
- CSS3
- Node.js
- Express.js
- Nodemailer
- Git & GitHub
